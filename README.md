# PBL_PrakAlpro_MoneyTracker_Kelompok1
Sistem ini dibuat untuk membantu pengguna mencatat dan merekap seluruh pengeluaran harian secara 
terstruktur. Melalui aplikasi ini, pengguna dapat memasukkan data pengeluaran seperti makan, transport, 
dan kebutuhan lainnya beserta nominal uang yang dikeluarkan.
